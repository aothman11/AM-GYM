# AM-Gym 🏋️

> Train Smart. Eat Right. — Free fitness companion with exercise guides and global nutrition data.

## Deploy to Vercel (3 steps)

1. **Upload to GitHub** — push this folder to a new GitHub repo
2. **Connect Vercel** — go to [vercel.com](https://vercel.com), click "Add New Project", import your GitHub repo
3. **Deploy** — Vercel auto-detects the static site. Click Deploy. Done ✓

Your app will be live at `your-project.vercel.app` in ~30 seconds.

---

## What's inside

| File | Purpose |
|------|---------|
| `index.html` | The entire app (HTML + CSS + JS) |
| `vercel.json` | Vercel routing & security headers |
| `package.json` | Project metadata |

## Features

- **20+ exercises** with real YouTube video embeds
- **40+ global meals** — Saudi, Mediterranean, Asian, American, Mexican
- Macro breakdowns + healthy swap tips for every meal
- Filter by muscle group (chest, back, legs, shoulders, arms, core)
- Filter meals by cuisine
- Mobile-first design with bottom tab navigation
- Zero data collection — no login, no cookies, no tracking

## Local dev

```bash
npx serve .
```
